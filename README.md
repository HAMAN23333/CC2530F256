# 使用手册

>该项目为Zigbee门禁开发 
>
>开发环境基于[IAR Embedded Workbench for 8051](https://www.iar.com/products/architectures/iar-embedded-workbench-for-8051/#containerblock_3094)
>
>无需配置`basic_rf`包，已自动部署

- 红外反射传感器
